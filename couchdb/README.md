# CouchDB databáze

## Konfigurace
Obsah `.env` souboru:
```
# Uživatelské jméno pro plný přístup (admin).
COUCHDB_USER=
# Heslo
COUCHDB_PASSWORD=
```

## Datová úložiště
Data jsou uložena do adresáře `/opt/couchdb/data`.

## Logování
Logování do souboru není nastaveno a tak jsou logy zapisovány pouze na standardní výstup.

## Porty
- 5984
